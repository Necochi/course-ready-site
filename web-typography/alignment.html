Выравнивание Текста

Выравнивание По Левому Краю

Текст с выравниванием по левому краю является стандартом, поскольку он дает читателю вертикальную опору, к которой он может возвращаться на каждой строке. Вы должны использовать его для большей части основного текста (если у вас нет действительно веской причины этого не делать).

Выравнивание По Центру

Текст, выровненный по центру, не имеет такой опоры, поэтому глазам легче потеряться, когда они пытаются перейти на следующую строку. Используйте его для особых видов контента, таких как стихи, песни и заголовки.

Выравнивание По Правому Краю

Используйте текст, выровненный по правому краю, только если это имеет смысл для макета. Например, подпись к изображению:

Конечно, если вы публикуете контент на языке, который читается справа налево, это уже другая история...

Выравнивание По Ширине

Выровненный по ширине текст требует хорошего механизма переноса слов. Без интеллектуального способа разделения длинных слов по строкам появляются большие пробелы между словами, что затрудняет восприятие текста. Большинство браузеров не оснащены качественными встроенными механизмами переноса.
